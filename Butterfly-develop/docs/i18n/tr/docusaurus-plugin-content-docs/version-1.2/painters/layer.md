## Yapılandırma

|          Özellik | Varsayılan | Tanım                                                                                      |

| Silgiyi dahil et |   false    | Bu ayar, silgi katmanına tıklandığındaö onu düzenleyip düzenleyemeyeceğinizi kontrol eder. |
