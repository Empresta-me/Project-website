## 7 (Αλλαγή αλλαγής) {#7}

### Σπάζοντας τις αλλαγές

Το στρώμα γόμα θα αφαιρεθεί. Κατά τη μετανάστευση, το στρώμα γόμα θα αφαιρεθεί.
