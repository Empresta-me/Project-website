::note ⚡ Tablica akcji

Jest to szczególny malarz. Nie możesz go wybrać i uruchomi akcję, jeśli klikniesz na nią.

:::

Po kliknięciu tej farby można powtórzyć ostatnią czynność. Zobacz [cofnij farbę](undo) aby cofnąć ostatnią akcję.
