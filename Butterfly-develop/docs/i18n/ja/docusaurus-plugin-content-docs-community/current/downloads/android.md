[このリンク](https://play.google.com/apps/testing/dev.linwood.butterfly) をクリックして、アプリの [ナイトリー](/nightly) バージョンに登録してください。 このバージョンにはバグがありますのでご注意ください。 プレイストアのバージョンは、新しいナイトリーバージョンがリリースされると自動的に更新されます。
