この画家でこの紙に描けます。

## 設定

|      属性 | デフォルト | 説明                         |
| -------:|:-----:|:-------------------------- |
|       色 | ブラック  | 描画される色                     |
| ストロークの幅 |   5   | ストロークの幅                    |
| ストローク乗数 |   1   | スタイラスを持っている場合、これは加算される乗数です |
