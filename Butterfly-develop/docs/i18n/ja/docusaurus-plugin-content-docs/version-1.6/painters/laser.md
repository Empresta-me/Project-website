この画家では一時的にこの紙に描けます。 しばらくすると、絵は薄れていきます。 キャンセルするために、新しい図面を紙に追加します。

## 設定

|      属性 | デフォルト | 説明                         |
| -------:|:-----:|:-------------------------- |
|       色 | ブラック  | 描画される色                     |
| ストロークの幅 |   5   | ストロークの幅                    |
| ストローク乗数 |   1   | スタイラスを持っている場合、これは加算される乗数です |
|      期間 |   5   | 図面が表示されるまでの時間 (秒)          |
