:::warning 消しゴムを使うことは推奨されません。

可能であれば、 [パス消しゴム](path_eraser) を使用してください。

:::

この画家で紙の要素を消すことができます。

## 設定

|      属性 | デフォルト | 説明                         |
| -------:|:-----:|:-------------------------- |
| ストロークの幅 |   5   | ストロークの幅                    |
| ストローク乗数 |   1   | スタイラスを持っている場合、これは加算される乗数です |
