:::警告：不建议使用擦除

如果可能，请使用 [路径擦除](path_eraser)。

:::

通过这个绘制器，你可以擦除纸上的元素。

## 配置

|   财产 | 默认设置 | 描述                     |
| ----:|:----:|:---------------------- |
| 笔画宽度 |  5   | 笔画的宽度                  |
| 笔记倍数 |  1   | 如果您有一个样式表，这是将添加到压力中的乘数 |
