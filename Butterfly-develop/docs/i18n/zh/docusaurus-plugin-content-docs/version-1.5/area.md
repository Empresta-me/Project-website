## 一. 导言

![区域](area.png)

区域是限制画布面积的一种方式。 您可以通过使用 [区域绘制器](painters/area.md) 来创建区域。 如果您点击后台，您可以使用上下文菜单打开区域列表对话框。

## 创建和删除区域

使用 [区域画面](painters/area.md) 来创建一个区域。 领域不能重叠。 在你创建了一个区域后，你需要给它命名。

如果你想要修改它，请单击该区域。 您也可以使用区域对话框。

您在此菜单中有几个选项：

* `输入区域`: 输入区域。 你不能在区域外涂抹。 通过点击带画面的背景退出区域。
* `名称`: 更改当前区域的名称
* `导出`: 画布的这一部分导出为图像
* `删除`: 删除此区域
