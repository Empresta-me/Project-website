## Configuratie

|              Eigenschap | Standaard | Beschrijving                                                                      |
| -----------------------:|:---------:|:--------------------------------------------------------------------------------- |
|          Stroke breedte |     5     | De breedte van de beroerte                                                        |
| Stroke vermenigvuldiger |     1     | Als je een stijl hebt, is dit de multiplier die aan de druk zal worden toegevoegd |
