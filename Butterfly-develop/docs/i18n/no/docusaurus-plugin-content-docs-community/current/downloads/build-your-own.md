---
title: "Bygg din egen"
sidebar_position: 1
---

1. Installere git og flutter (beta)
2. Klon kodelageret
3. Naviger til app mappen
4. Bruk fløderverktøyet for å kompilere applikasjonen
   * `flutter build apk`
   * `flutter build appbundle`
   * `flutter build web`
   * `flutter build linux`
   * `flutter build windows`
5. De kompilerte filene finnes i byggemappen
