الحزمة هي مجموعة من المكونات التي يمكن استخدامها في لوحة الطوابع. يمكنك إنشاء حزمة خاصة بك أو استيراد حزمة موجودة. لهذا انقر فوق الزاوية اليسرى العلوية وانقر على "الحزم".

## الخصائص

|  الاسم | الوصف       |
| ------:|:----------- |
|  الاسم | اسم الحزمة  |
| المؤلف | مؤلف الحزمة |
|  الوصف | وصف للحزمة  |
