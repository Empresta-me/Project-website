يمكنك بدء التطبيق بالحجج.

`<Path> butterfly.exe`

المسار هو المسار إلى الملف الذي تريد فتحه.

## خيارات

|             خيار | الوصف                                                 |
| ----------------:|:----------------------------------------------------- |
| `--المسار`، `-ب` | اختر دليل البيانات حيث يتم تخزين جميع بيانات الفراشة. |
