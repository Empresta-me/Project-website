إنشاء رسم جديد بالنقر على أيقونة زائد. تغيير ترتيب الرسام عن طريق السحب وتغيير الرسام بالنقر. إذا قمت بالنقر على الرسام المحدد يمكنك تكوينه.

سيتم عرض اسم الرسام كتلميح على لوحة الأدوات على الرسم.
