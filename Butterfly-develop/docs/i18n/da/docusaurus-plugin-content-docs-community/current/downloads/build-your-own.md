title: "Byg din egen"
sidebar_position: 1
---

1. Installer git og flutter (beta)
2. Klon depotet
3. Navigér til app-mappen
4. Brug flagerværktøjet til at kompilere programmet
   * `flutter build apk`
   * `flutter build appbundle`
   * `flutter build web`
   * `flutter build linux`
   * `flutter build windows`
5. De kompilerede filer er i byggemappen
